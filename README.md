# **.dotfiles**

## Installing the Xcode Command Line Tools on a Mac
```zsh
xcode-select --install
```

## Home Brew
> 1. Install Brew  

```zsh
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```
> 2. Install bundle

```zsh
cd ~/.dotfiles && brew bundle
```
